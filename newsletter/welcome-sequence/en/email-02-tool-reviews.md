## Review Framework

1. **Real-project testing**—We do not run demos; we validate on the actual PeterClaw codebase
2. **Multi-dimensional scoring**—Code completion, agent capabilities, context understanding, response speed, and pricing
3. **Scenario-based recommendations**—No absolute "winner"; we tell you which tool fits which situation
