# 内容跨平台发布检查清单

> 目标：确保每次网站文章发布后，30 分钟内完成全平台标准化分发
> 适用平台：微信公众号、知乎专栏、掘金、V2EX 独立开发者、少数派
> 维护者：Kimi 1号（内容策略）
> 最后更新：2026-05-28

---

## Phase 0：网站发布（前置条件）

- [ ] `draft: false` 已设置
- [ ] `npm run quality:content` 通过
- [ ] `npm run build` 通过
- [ ] 网站已部署，文章可访问
- [ ] 原文链接确认：`https://peterclaw.com/zh/blog/{slug}/`

---

## Phase 1：掘金（预计 5 分钟）

- [ ] 标题符合掘金风格（≤ 50 字符，可带 emoji，突出技术点）
- [ ] 开头有「阅读约 X 分钟」提示
- [ ] 原文链接已放在开头引用块
- [ ] 代码块已标注语言
- [ ] 表格保留，格式正确
- [ ] 图片已上传掘金图床
- [ ] 文末有「关于作者」区块（GitHub + Twitter + Newsletter）
- [ ] 文末有「相关阅读」链接
- [ ] 分类和标签已选（最多 5 个标签）
- [ ] **已发布**

---

## Phase 2：知乎专栏（预计 5 分钟）

- [ ] 标题符合知乎风格（≤ 50 字符，问题感或方法论）
- [ ] 首段前有 TL;DR（≤ 100 字）
- [ ] 表格保留，加文字导读
- [ ] 代码块保留，加语言标注
- [ ] 外链已判断可信度，必要时加 `[非官方来源]` 提示
- [ ] 同一篇文章外链 ≤ 8 条
- [ ] 文末有 `本文首发于 PeterClaw 网站，转载请联系授权`
- [ ] 文末有 1–2 个开放讨论问题
- [ ] 话题已绑定（人工智能 / AIGC / 软件工程 / 团队协作 / 个人开发者）
- [ ] **已发布**

---

## Phase 3：少数派（预计 5 分钟，审核非即时）

- [ ] 标题符合少数派风格（≤ 40 字符，品质感）
- [ ] 首段说明「适合谁 + 读完带走什么」
- [ ] 封面图已准备（1200×630，视觉精美）
- [ ] 内文图片已上传少数派图床
- [ ] 段落简短（手机端 ≤ 4 行/段）
- [ ] 工具链描述完整（用什么 + 为什么 + 怎么配合）
- [ ] 有具体数字（效率提升 X%、时间节省 X 小时）
- [ ] 文末有作者介绍和社交平台链接
- [ ] 分类已选（效率工具 / 编程与开发 / 数字生活）
- [ ] **已提交审核**

---

## Phase 4：V2EX（预计 3 分钟）

- [ ] 标题像帖子（分享型/问题型/数据型）
- [ ] 字数在 150–400 字之间
- [ ] 第一段一句话说清文章价值
- [ ] 2–3 个核心观点/数据点已列出
- [ ] 原文绝对链接已包含（非短链接）
- [ ] 至少 1 个开放讨论问题
- [ ] 语气像聊天，不是推送
- [ ] 发布到 `creators` 节点
- [ ] **已发布**

---

## Phase 5：微信公众号（预计 10 分钟，耗时最长）

- [ ] 标题 ≤ 64 字符，含情绪词或数字
- [ ] 首段 ≤ 150 字，快速给出核心价值
- [ ] 正文所有 URL 已删除，改为纯文字
- [ ] 文末「延伸阅读」集中列出 ≤ 5 条链接
- [ ] Markdown 表格已转为文字描述或简化列表
- [ ] 代码块已截图或简化描述
- [ ] 图片已上传微信素材库
- [ ] 每 3–4 段有视觉分隔
- [ ] 文末有作者简介 + 引导关注 + 相关阅读卡片
- [ ] 文章标签：`AI`、`软件开发`、`团队协作`、`个人成长`
- [ ] 已加入「AI 小队组建日记」合集（如适用）
- [ ] 预览检查排版无错位
- [ ] **已定时发送**（选择最佳时段）

---

## Phase 6：发布后动作（预计 5 分钟）

- [ ] 在网站原文评论区置顶同步信息
- [ ] 记录各平台发布链接到 Notion/文档（用于追踪）
- [ ] 掘金：记录阅读/点赞/评论数（24h 后复查）
- [ ] 知乎：记录赞同/收藏比（24h 后复查）
- [ ] V2EX：跟踪讨论回复，及时参与关键评论
- [ ] 少数派：确认审核状态（通常 1–3 个工作日）
- [ ] 微信公众号：记录阅读完成率（24h 后复查）

---

## 时间预算总览

| 阶段 | 平台 | 预计时间 | 可否并行 |
|------|------|----------|----------|
| 0 | 网站发布 | 3–5 分钟 | — |
| 1 | 掘金 | 5 分钟 | ✅ 与知乎/V2EX 并行 |
| 2 | 知乎 | 5 分钟 | ✅ 与掘金/V2EX 并行 |
| 3 | 少数派 | 5 分钟 | ✅ 与掘金/知乎/V2EX 并行 |
| 4 | V2EX | 3 分钟 | ✅ 与掘金/知乎/少数派 并行 |
| 5 | 微信公众号 | 10 分钟 | ❌ 需单独排版 |
| 6 | 发布后动作 | 5 分钟 | ❌ 需等平台数据 |
| **总计** | | **约 30 分钟** | |

**并行策略**：先同时打开掘金、知乎、少数派、V2EX 四个标签页，复制粘贴适配版本。完成后统一发布。最后处理微信公众号排版。
