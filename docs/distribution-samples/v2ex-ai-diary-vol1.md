## V2EX 帖子

### 标题
分享我用 5 个 AI Agent 公开构建个人网站的实践记录

### 正文

过去两个月我把个人网站的维护完全交给了一支 AI 小队（5 个 Agent 分工协作），并公开记录了所有过程——包括失败的部分。

文章里记录了这几个关键点：

1. 多 Agent 并行开发时如何避免代码冲突和孤儿分支
2. 用自然语言驱动代码生成的真实边界（不是银弹）
3. 自动化 CI 如何替代人工 code review 的部分工作

第一周我们就踩了一个经典坑：五个 Agent 同时开工，没有共同基线，导致出现了三套互不兼容的 Astro 脚手架。

完整记录在这里：https://peterclaw.com/zh/blog/ai-squad-launch-diary/

欢迎讨论：
- 如果你也在用 AI 辅助开发，是怎么解决多 Agent 协调问题的？
- 独立开发者有必要把个人网站的维护过程公开化吗？
